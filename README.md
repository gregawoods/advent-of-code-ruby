# Advent of Code

Ruby solutions to [Advent of Code](https://adventofcode.com/).

## Requirements

Install ruby 3.1.3, bundler, and the gems.

```
bundle install
```

## Use

```
# solve all
rake solve:all

# solve a specific day
rake solve:7

# run all tests
rake test

# run a specific test
ruby ./test/07_test.rb
```
